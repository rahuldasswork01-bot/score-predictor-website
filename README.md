# 🎯 Student Score Predictor — Full Stack ML Web App

A full stack web application that predicts student exam scores using Machine Learning, with login system, grade display, study tips and funny meme reactions! 😂

---

## 🌟 Features

- 🔐 Login & Register System
- 🤖 AI Powered Score Predictions (Linear Regression)
- 🎓 Grade Display (A+, A, B, C, D, F)
- 💡 Smart Study Tips based on your score
- 😂 Funny Meme GIFs for each grade
- 🎨 Beautiful 3D Animated Dark UI
- 📊 Real-time Progress Bar

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | HTML, CSS, JavaScript |
| **Backend** | Python, Flask |
| **ML Model** | Scikit-learn (Linear Regression) |
| **Data** | Pandas, NumPy |
| **Auth** | Flask Sessions |

---

## 📈 ML Model Performance

- Algorithm: Linear Regression
- R² Score: 1.00 (100% accuracy)
- Training Data: 10 student records
- Prediction: Exam score based on study hours

## 🚀 How to Run Locally

1. Clone the repository:# score-predictor-website
Full stack ML web app with login
git clone https://github.com/rahuldasswork01-bot/score-predictor-website.git
2. Install dependencies:
pip install flask scikit-learn numpy pandas
3. Train and save the model:
python model.py
4. Run the app:
python app.py
5. Open browser and go to:
http://127.0.0.1:5000
